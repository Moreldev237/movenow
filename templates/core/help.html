Réservation

Pour réserver une course:

  1. Ouvrez l'application MoveNow
  2. Entrez votre destination
  3. Choisissez votre type de véhicule
  4. Sélectionnez un chauffeur
  5. Confirmez votre réservation

Oui! Vous pouvez programmer une course jusqu'à 7 jours à l'avance. Allez dans l'onglet "Programmer" et choisissez la date et l'heure souhaitées.

Vous pouvez annuler une réservation depuis l'écran de suivi. Notez que des frais d'annulation peuvent s'appliquer selon les conditions.

Le prix est calculé à l'avance et reste généralement stable. Cependant, des suppléments peuvent s'appliquer en cas de détour significatif ou de temps d'attente prolongé.

Paiement

MoveNow accepte:

  • Espèces
  • Mobile Money (Orange Money, MTN Mobile Money)
  • Cartes bancaires (Visa, Mastercard)
  • Solde MoveNow

Pour les paiements numériques, les remboursements sont traités sous 5-7 jours ouvrés.Contactez le support via l'application.

Entrez votre code promo dans le champ prévu à cet effet lors de la confirmation de votre réservation. La réduction sera automatiquement appliquée.

Compte

Télécharger l'application MoveNow et suivez les instructions. Vous aurez besoin d'un numéro de téléphone valide pour vous inscrire.

Sur l'écran de connexion, cliquez sur "Mot de passe oublié" et suivez les instructions envoyées par SMS ou email.

Allez dans Paramètres > Compte > Supprimer le compte. Cette action est irréversible.

Sécurité

Tous nos chauffeurs passent par un processus de vérification strict incluant:

  • Vérification du permis de conduire
  • Contrôle du casier judiciaire
  • Vérification des documents du véhicule
  • Entretien avec notre équipe

Si vous rencontrez un problème:

  1. Utilisez le bouton SOS dans l'application
  2. Contactez le support au +237 686 865 451
  3. Signalez l'incident via l'historique de courses
